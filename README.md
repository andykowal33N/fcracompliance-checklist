# FCRA Compliance Checklist

Practical checklist for ensuring compliance with the Fair Credit Reporting Act (FCRA) in continuous vetting and due-diligence workflows.

## Overview
This repository provides a structured, practical framework for managing FCRA compliance. It’s designed for analysts, developers, and compliance officers building systems that process or analyze personal or financial data as part of screening, vetting, or investigative operations.

## Key Sections
1. **Permissible Purpose** — Establish and document the lawful basis for each use of consumer data.  
2. **Disclosure & Authorization** — Implement separate written consent before collecting or analyzing data.  
3. **Adverse Action** — Define pre- and post-adverse notice procedures.  
4. **Dispute & Reinvestigation** — Create workflows for consumer dispute handling.  
5. **Data Handling & Retention** — Implement access control and secure destruction policies.  
6. **Vendor Oversight** — Ensure agreements and audits for third-party providers.

> **Note:** This content is for operational reference only and not legal advice.
